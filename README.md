Explore perguntas sobre países, idiomas, população e curiosidades do mundo enquanto avança pelas telas interativas do jogo.

Criado inteiramente no PowerPoint, o projeto transforma apresentações em um jogo, permitindo navegar entre perguntas, descobrir respostas e avançar até o final.

Além das perguntas, o jogo conta com efeitos sonoros, música de fundo e alguns segredos escondidos para quem decidir explorar um pouco mais.

Recomendado jogar com som ligado ou usando fone de ouvido.

> 🚫 Nenhuma IA generativa foi utilizada na criação deste jogo.
